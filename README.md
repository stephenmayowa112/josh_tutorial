# Josh Tutorials

An introductory web development project for learning the basics of HTML and CSS.

## What This Project Shows

This project is a simple static web page that demonstrates:

- Basic HTML structure
- Headings from `h1` to `h6`
- Paragraph text and emphasis tags
- Simple CSS styling with background colors

## Files

- `index.html` - Main webpage content
- `style.css` - Basic styling for the page
- `script.js` - JavaScript file reserved for future practice

## How to Run

1. Open `index.html` in a web browser.
2. Edit the HTML or CSS files to practice making changes.
3. Refresh the browser to see your updates.

## Purpose

This project is meant for beginners who are learning how HTML structures a page and how CSS changes its appearance.
